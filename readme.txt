== Description ==

`Installez notre extension et commencez à générer des leads en quelques minutes !`

Cette extension vous proposera l'insertion dans votre siteweb de 2 widgets au choix :

1. Chiffrage / Demande de visite d'entreprises : apporte un service unique à vos visiteurs et fidélise votre trafic, avec un temps moyen d'utilisation supérieur à 7 min.  

2. Demande de devis rapide : entièrement responsive, il présente un taux de transformation record. Il peut être paramétré très simplement pour s’ouvrir par défaut sur un formulaire donné, selon la page où il est affiché (exemple : demande de devis parquet proposé sur votre article traitant de parquets).

`Générez des demandes de devis et gagnez de l'argent`

Dès le widget installé, vous retrouvez dans wordpress un tableau statistique des leads générés avec un suivi de votre rémunération en temps réel !

Nous restons à votre disposition pour vous aider à maximiser vos revenus. Vous pouvez notamment utiliser nos kits email + landing page pour vos campagnes emailing.

== Installation ==

How to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/lead-generator-by-enchantier` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Générer Leads screen to configure the plugin
4. You will have to register a lead provider account on our system to track the leads you will generate.
